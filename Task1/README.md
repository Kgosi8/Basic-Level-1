3. **Open index.html** in your browser to view the website

## Customization Guide

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #4F46E5;     /* Main brand color */
    --primary-dark: #4338CA;       /* Darker shade */
    --text-dark: #1F2937;          /* Main text color */
    --text-light: #6B7280;         /* Secondary text */
}
```

### Content
- Edit text directly in the HTML files
- Replace placeholder images in the images folder
- Update contact information in `contact.html`

### Fonts
The website uses Google Fonts (Inter). To change:
1. Visit [Google Fonts](https://fonts.google.com/)
2. Select your preferred font
3. Replace the font link in the `<head>` section
4. Update `font-family` in CSS
